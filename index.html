Third section

Difference between paddings, margins and borders

The CSS padding properties are used to generate space around an element's content, inside of any defined borders. The CSS margin properties are used to create space around elements, outside of any defined borders. The CSS border properties allow you to specify the style, width, and color of an element's border.
